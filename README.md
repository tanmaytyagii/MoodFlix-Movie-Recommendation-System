# 🎬 MoodFlix

Sentiment-Based Movie Recommendation System

## 🚀 Overview

MoodFlix is a movie recommendation system that suggests movies based on user sentiment using NLP techniques and real-time data from TMDB API.

---

## 🛠️ Tech Stack

* React (TypeScript)
* Vite
* Tailwind CSS
* Python (NLP)
* TMDB API

---

## ⚙️ Run Locally

```bash
git clone https://github.com/tanmaytyagii/MoodFlix-Movie-Recommendation-System.git
cd MoodFlix-Movie-Recommendation-System
npm install
npm run dev
```

---

## 🧠 Features

* Sentiment-based movie recommendations
* Real-time movie data (TMDB API)
* Clean and responsive UI
